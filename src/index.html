AdoptACat

AdoptACat is a web application made using React, JSX, HTML, CSS, and Bootstrap. Giving users access to over 1k+ cats in shelters ready for adoption, AdoptACat provides an easy user interface for adoption seekers. AdoptACat allows users to favorite cats to their local storage so they can view their cats of interest at anytime and update it as well as needed. Users can also narrow their search through the use of filters such as gender, breed, any many more.

idetect

idetect is a full stack web application that utilizes the Clarifai face detection API to detect all faces in a photo. After a succesful url input to any photo on the web, if a face is detected, the inputted photo will be displayed with bounding boxes around the detected faces. The front-end side of this application was made using React. The back-end RESTful API server was made using Node.js and Express.js. The database that contains user information was made using PostgeSQL. idetect has been deployed on Heroku since January of 2022.

VillagerVerse

VillagerVerse is a web application made using React and React Hooks. that utilizes the ACNH RESTful API. This application is a gallery of all of the characters from the video game Animal Crossing: New Horizons. The search bar feature allows you to search for a specific villager and the display will filter and update as you type. Clicking on a specific villager card will display more details about the villager.

Cozy Cat Home

Cozy Cat Home is a mobile game application where you can adopt, care for, and interact with cats. Clicking on a cat will display a pop-up that shows their stats such as their hunger level. You can manage their hunger level by purchasing food items at the store. To earn coins to feed your cats, simply clean any pawprints you may see on the floor by clicking on it. Purchased items will go to your inventory where you can drag and drop items to any cat.
